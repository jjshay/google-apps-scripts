## [1.0.0] - 2025-01-11

### Added

#### 3DSellers Integration Scripts
- 3DSELLERS_V7_MASTER.gs: Flagship inventory management with Claude Vision AI analysis
- 3DSELLERS_V5_FIXED.gs: Stable version with core inventory features
- 3dsellers-production-ready.gs: Production-optimized 3DSellers integration
- UNIFIED_3DSELLERS_V3_COMPLETE.gs: Complete unified 3DSellers system
- FINAL_UNIFIED_3DSELLERS.gs: Final consolidated 3DSellers automation
- 3D_SELLERS_IMAGE_CROPPING.gs: Image cropping automation for product photos
- CREATE_CONFIG_SHEET.gs: Configuration wizard for setting up CONFIG sheets

#### AI Integration Scripts
- AI-ENHANCED-GOOGLE-APPS-SCRIPT.gs: General-purpose AI integration with OpenAI and Gemini
- AI-METADATA-ENHANCED-SCRIPT.gs: AI-powered metadata extraction from images
- AI_ART_SORTER.gs: Automatic artwork sorting into artist-specific folders using Claude Vision
- ROBUST_Dual_AI_Checker_Script.gs: Dual-AI verification using Claude and GPT-4

#### News Engine Scripts
- NEWS_AudienceWeighting.gs: AI-powered article scoring by audience relevance
- NEWS_FeedlyEmail_SlideDeck.gs: Convert Feedly email digests to Google Slides
- NEWS_Pipeline_UI.gs: User interface for news processing pipeline

#### Utility Scripts
- CREATIVE-AUTO-RENAMER.gs: AI-powered file renaming based on image content
- ENHANCED-SALES-CHANNEL-ANALYZER.gs: Multi-channel sales analysis for eBay, Etsy, Poshmark, and StockX
- EbayAutomation.gs: SKU generation and listing automation for eBay
- FRAME_VARIATIONS_SIMPLE.gs: Frame variation listing generation
- GALLERY-GAUNTLET-AUTO-FORMAT-SCRIPT.gs: Gallery data formatting automation
- GoogleAppsScript_ChannelSeparator.gs: Data separation by sales channel
- Google_Apps_Script_INBOUND_Monitor.gs: Folder monitoring with alerts
- drive_folder_catalog.gs: Google Drive folder cataloging
- google_sheets_uploader.gs: Image upload from Google Drive to external servers
- populate_image_urls_v3.gs: Image URL population in spreadsheet cells

#### Core Features
- Multi-AI support for Claude, GPT-4, and Gemini APIs
- Automatic SKU generation with smart naming conventions
- Artist-specific pricing rules
- Batch processing with progress tracking and error recovery
- Hourly trigger support for automated monitoring
- Custom menu creation patterns
- API call retry logic with exponential backoff
- Configuration management via CONFIG sheets
